# vanguardeira-project

Infrastructure and educational technology project developed in partnership with the Escola de Sargentos das Armas (ESA).

The project focused on transforming seized and donated TV Box devices into fully functional Linux-based educational workstations.

## Overview

Vanguardeira was created to promote:

* Technological reuse
* Sustainable computing
* Low-cost infrastructure
* Educational accessibility
* Embedded Linux deployment

More than 60 TV Box devices were converted into operational Linux systems for educational and laboratory environments.

## Areas of Participation

* Linux deployment
* Embedded operating systems
* Network configuration
* Infrastructure support
* Hardware preparation
* Security configuration
* System testing and validation

## Technologies

* Linux
* Armbian
* ARM architecture
* Embedded systems
* Networking
* Self-hosted infrastructure

## Project Goals

* Reuse low-cost hardware
* Create stable educational workstations
* Reduce infrastructure costs
* Promote sustainable technology practices
* Expand access to computing environments
